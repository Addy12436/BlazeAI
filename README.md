# 🔥 Blaze AI – Website Generator using Spring Boot + Gemini + Unsplash + Razorpay

Blaze AI is a smart website generator built with **Java Spring Boot**, **Thymeleaf**, **Gemini API**, **Razorpay**, and **Unsplash**. It allows users to enter prompts and instantly generate modern, responsive websites using AI.

---

## ✨ Features

- 🔐 User Login/Signup (via Email or Google)
- 💬 AI-generated HTML websites using Gemini API
- 🖼️ Realistic images from Unsplash API
- 💳 Razorpay integration to go Pro
- 🔒 Free user daily generation limit (Pro = unlimited)
- 💡 Download or preview the generated website
- 📱 Responsive frontend with Tailwind CSS
- 🔥 Beautiful animated loader while generating

---

## 📁 Technologies Used

| Layer       | Stack / Tool                       |
|-------------|------------------------------------|
| Backend     | Spring Boot, Java 17               |
| Frontend    | HTML5, CSS3, Thymeleaf, JavaScript |
| AI Service  | Gemini API                         |
| Images      | Unsplash API                       |
| Payments    | Razorpay API                       |
| Database    | MySQL                              |
| Auth        | Spring Security + Google OAuth     |
| Build Tool  | Maven                              |
